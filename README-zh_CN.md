# Iris web框架的Swagger中间件

[Iris](https://github.com/kataras/iris) 中间件，用于按照 [#1231](https://github.com/kataras/iris/issues/1231) 要求自动生成 RESTful API 文档，使用 Swagger 2.0 规范。

[![build status](https://img.shields.io/github/actions/workflow/status/iris-contrib/swagger/ci.yml?branch=master&style=for-the-badge)](https://github.com/iris-contrib/swagger/actions)
[![Go Report Card](https://goreportcard.com/badge/github.com/iris-contrib/swagger?style=for-the-badge)](https://goreportcard.com/report/github.com/iris-contrib/swagger)

## 使用方法

### 开始使用

1. 为您的 API 源代码添加注释，[查看声明式注释格式](https://swaggo.github.io/swaggo.io/declarative_comments_format/)。
2. 下载适用于 Go 的 [Swag](https://github.com/swaggo/swag) 工具：

```sh
$ go install github.com/swaggo/swag/cmd/swag@latest

# 如果发现 swag cli 无法工作，您可以尝试从源码安装 swag cli
git clone git@github.com:swaggo/swag.git
cd swag
# tag 变量应与 go.mod 中的 github.com/swaggo/swag 匹配
# 这里我们使用 v1.8.10
git checkout -b ${tag} tags/${tag}
go install ./cmd/swag
```

3. 在包含 `main.go` 文件的 Go
   项目根文件夹中运行 [Swag](https://github.com/swaggo/swag)，[Swag](https://github.com/swaggo/swag) 将解析注释并生成所需文件（
   `docs` 文件夹和 `docs/doc.go`）。

```sh
$ swag init
```

4. 下载 [Iris 版 Swagger](https://github.com/iris-contrib/swagger)：

```sh
$ go get github.com/iris-contrib/swagger/v12@master
```

并在代码中导入以下包：

```go
import "github.com/Desdemo/fkiris-swagger"              // swagger middleware for Iris
import "github.com/Desdemo/fkiris-swagger/swaggerFiles" // swagger embed files

```

### 示例代码：

```go
package main

import (
	"github.com/8treenet/iris/v12"

	"github.com/Desdemo/fkiris-swagger"
	"github.com/Desdemo/fkiris-swagger/swaggerFiles"

	_ "github.com/your_username/your_project/docs"
	// docs folder should be generated by Swag CLI (swag init),
	// you have to import it.
)

// @title Swagger 示例 API
// @version 1.0
// @description 这是一个示例 Petstore 服务器。
// @termsOfService http://swagger.io/terms/

// @contact.name API 支持
// @contact.url http://www/swagger.io/support
// @contact.email support@swagger.io

// @license.name Apache 2.0
// @license.url http://www.apache.org/licenses/LICENSE-2.0.html

// @host localhost:8080
// @BasePath /v2
func main() {
	app := iris.New()

	swaggerUI := swagger.Handler(swaggerFiles.Handler,
		swagger.URL("/swagger/doc.json"),
		swagger.DeepLinking(true),
		swagger.Prefix("/swagger"),
	)

	// 注册到 http://localhost:8080/swagger
	app.Get("/swagger", swaggerUI)
	// 以及通配符路由用于 index.html, *.js, *.css 等文件
	app.Get("/swagger/{any:path}", swaggerUI)

	app.Listen(":8080")
}
```

5. 运行它，并通过 <http://localhost:8080/swagger/index.html> 访问，您应该能看到 Swagger 2.0 API 文档页面。

6. 如果您想在设置某个环境变量时禁用 swagger，可以使用 `DisablingHandler` 代替 `Handler`。

```go
swagger.DisablingHandler(swaggerFiles.Handler, "THE_OS_VARIABLE_NAME_HERE", configurators ...Configurator)
```

7. 如果您想更改 swagger-ui 主题，可以在初始化 `swaggerUI` 时添加 `swagger.SetTheme(swagger.Monokai)`

```go
swaggerUI := swagger.Handler(swaggerFiles.Handler,
swagger.URL("/swagger/doc.json"),
swagger.DeepLinking(true),
swagger.Prefix("/swagger"),
// 参考: https://github.com/ostranme/swagger-ui-themes
// 当前支持 7 个主题
// 主题是可选配置，如果不设置，将使用默认主题
swagger.SetTheme(swagger.Monokai),
)
```

## 高级配置

### 多个Swagger文档

您可以使用 `URLs` 配置来支持多个Swagger文档：

```go
swaggerUI := swagger.Handler(swaggerFiles.Handler,
swagger.URLs("/swagger/v1/doc.json", "/swagger/v2/doc.json"),
swagger.Prefix("/swagger"),
)
```

### OAuth2 集成

支持完整的OAuth2配置：

```go
swaggerUI := swagger.Handler(swaggerFiles.Handler,
swagger.URL("/swagger/doc.json"),
swagger.OAuth(&swagger.OAuthConfig{
ClientId: "your-client-id",
Realm:    "your-realm",
AppName:  "Your App",
}),
swagger.Oauth2UsePkce(true),
swagger.Oauth2DefaultClientID("your-default-client-id"),
)
```

### 其他高级选项

```go
swaggerUI := swagger.Handler(swaggerFiles.Handler,
swagger.URL("/swagger/doc.json"),
swagger.DeepLinking(true),
swagger.DocExpansion("list"),        // none, list, full
swagger.DefaultModelsExpandDepth(1), // -1 完全隐藏模型
swagger.PersistAuthorization(true),
swagger.SyntaxHighlight(true),
swagger.DomID("#custom-swagger-ui"),
swagger.Prefix("/api-docs"),
)
```

### 配置选项

| 配置函数                              | 类型             | 默认值                              | 描述                      |
|-----------------------------------|----------------|----------------------------------|-------------------------|
| `URL(url)`                        | `string`       | `"swagger/swagger.json"`         | API定义的URL               |
| `URLs(urls...)`                   | `[]string`     | `[]`                             | 多个API定义的URLs            |
| `Prefix(prefix)`                  | `string`       | `"/swagger"`                     | UI注册的前缀                 |
| `DeepLinking(deepLinking)`        | `bool`         | `true`                           | 启用深度链接                  |
| `DocExpansion(expansion)`         | `string`       | `"list"`                         | 文档展开模式 (none/list/full) |
| `DefaultModelsExpandDepth(depth)` | `int`          | `0`                              | 模型默认展开深度                |
| `PersistAuthorization(persist)`   | `bool`         | `false`                          | 持久化授权信息                 |
| `SyntaxHighlight(highlight)`      | `bool`         | `false`                          | 启用语法高亮                  |
| `OAuth(config)`                   | `*OAuthConfig` | `nil`                            | OAuth2配置                |
| `Oauth2UsePkce(usePkce)`          | `bool`         | `false`                          | 启用PKCE                  |
| `Oauth2DefaultClientID(clientId)` | `string`       | `""`                             | 默认OAuth2客户端ID           |
| `DomID(domID)`                    | `string`       | `"#swagger-ui"`                  | DOM元素ID                 |
| `Filter(filter)`                  | `bool`         | `true`                           | 启用标签过滤                  |
| `SetTheme(theme)`                 | `Theme`        | `Unknow`                         | UI主题                    |
| `FontCDN(cdn)`                    | `string`       | `"https://fonts.googleapis.com"` | 字体CDN                   |

### 支持的主题

- `FeelingBlue` - 感觉蓝色
- `Material` - 材质
- `Muted` - 柔和
- `Outline` - 轮廓
- `Flattop` - 平顶
- `Monokai` - Monokai
- `Newspaper` - 报纸
